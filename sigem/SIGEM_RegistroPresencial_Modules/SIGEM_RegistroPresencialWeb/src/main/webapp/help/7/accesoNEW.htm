Acceso al sistema

  

         Cambiar contraseña

Para llegar a la ventana de acceso del escritorio web de Registro SIGEM hay que indicar en el explorador la dirección URL y a continuación validarla:

                                                                         http://nombre de maquina ó IP/SIGEM_RegistroPresencialWeb

 

Lo siguiente que se ha de hacer es rellenar los datos de identificación y contraseña del usuario que se va a conectar al sistema, y después pulsar  para que se validen los datos de la conexión.

 

El sistema le da la posibilidad de guardar el nombre y la contraseña de cada usuario que se conecta con el fin de que cuando dicho usuario se conecte no tenga que escribir su contraseña ya que se asociará de forma automática.

Esta operación sólo se puede hacer la primera vez que se conecta un usuario. En este caso el sistema le pregunta si desea guardar los datos de identificación. Si contesta afirmativamente, cada vez que dicho usuario se vuelva a conectar el campo contraseña se rellenará automáticamente. Si no desea esta posibilidad basta con que conteste de forma negativa a la pregunta anterior.

Cambiar contraseña

Cualquier usuario que lo desee puede cambiar su contraseña. Tan sólo tiene que seleccionar esta opción y se abrirá otra ventana para introducir los nuevos datos.

 

Escriba el nombre de Usuario del sistema.

Escriba la Contraseña actual.

Escriba la Nueva contraseña.

Vuelva a escribirla en el campo Confirmación de nueva contraseña.

Pulse                                     

Si pulsa      se anulará la modificación de la contraseña.
