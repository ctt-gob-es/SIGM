Inicialmente en esta página aparecen las oficinas a las que el usuario ya ha sido agregado.
En este caso aún no ha sido agregado a ninguna otra.

Para agregar el usuario a una oficina existente, basta con pulsar el botón el botón. Añadir, elegir una del listado y pulsar el botón Aceptar.

Para eliminar la asociación entre un usuario y una oficina agregada se debe pulsar el botón Quitar que aparece junto a la oficina a la que se ha agregado el usuario.

Se puede volver al listado de usuarios pulsando el botón Cancelar.
