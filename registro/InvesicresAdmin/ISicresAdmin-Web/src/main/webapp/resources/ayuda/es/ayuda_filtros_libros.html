Desde está página se pueden editar los filtros tanto para usuarios como para oficinas. Mediante esta opción se permite que se tenga opción de consulta sobre una parte del libro de registro en lugar de que se tenga este permiso o no para la totalidad del libro.

A está zona se accede desde las opciones 'Aplicar Filtros' del listado de oficinas asociadas a un libro o desde el listado de usuarios de una oficina o agregados a esta. Ambos en el interior de la pestaña de Libros.

Para editar los filtros, tras rellenar el formulario pulse el botón Guardar para editar la oficina y volver al listado de oficinas o el botón Cancelar para ir al listado sin modificarla. También puede vaciar el formulario con el botón Borrar Todo, aunque no tendrá ninguna repercusión hasta que no pulse el botón Guardar.

Los botónes '+' y '-' permiten añadir nuevas filas o eliminarlas. Los campos de cada una de las filas son las siguientes:

  1. Campo: Dato del registro sobre el que se quiere aplicar el filtro.
  2. Operador: Tipo de filtro a realizar, igualdad, comenzar por, 'etc..'. La lista de valores dependerá del campo elegido.
  3. Valor: Valor con el que comparar, dependiendo del campo será texto libre, lista o diversos pop-ups.
  4. Nexo: Tipo de unión lógica con la fila anterior, el nexo de la última fila no debe rellenarse.
